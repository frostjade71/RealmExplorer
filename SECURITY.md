# Security Policy

We take the security of Realm Explorer seriously. If you believe you have found a security vulnerability in this project, please report it to us as described below.

## Supported Versions

We currently support the following versions of Realm Explorer with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v0.7.x  | ✅ Yes            |
| < v0.7  | ❌ No             |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them to us privately:

1. **Discord**: Send a direct message to a community leader on our [Discord](https://discord.gg/realmexplorer).
2. **Details to include**:
   - Description of the vulnerability.
   - Steps to reproduce.
   - Potential impact.
   - Any suggested fixes (if available).

## Our Commitment

- We will acknowledge your report within **48 hours**.
- We will work to investigate and resolve the issue as quickly as possible.
- We will keep you updated on our progress.
- Once fixed, we will provide credit to the researcher (if desired).

Thank you for helping keep the Realm Explorer community safe!
